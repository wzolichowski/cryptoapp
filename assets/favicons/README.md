# Favicons - Wymagane pliki

Ten folder zawiera wszystkie pliki favicon dla aplikacji "Kursy Walut".

## Lista wymaganych plików obrazów:

### Podstawowe favicony (przeglądarki):
- `favicon.ico` - 48x48px lub multi-size (16x16, 32x32, 48x48)
- `favicon-16x16.png` - 16x16px
- `favicon-32x32.png` - 32x32px
- `favicon-48x48.png` - 48x48px

### Apple Touch Icons (iOS/Safari):
- `apple-touch-icon.png` - 180x180px

### Android Chrome:
- `android-chrome-192x192.png` - 192x192px
- `android-chrome-512x512.png` - 512x512px

### Microsoft Windows Tiles:
- `mstile-70x70.png` - 70x70px (małe kafelki)
- `mstile-150x150.png` - 150x150px (średnie kafelki)
- `mstile-310x150.png` - 310x150px (szerokie kafelki)
- `mstile-310x310.png` - 310x310px (duże kafelki)

## Pliki konfiguracyjne (już utworzone):
- `site.webmanifest` - manifest dla Android/Chrome
- `browserconfig.xml` - konfiguracja dla Windows/IE/Edge

## Generowanie faviconów:

Możesz użyć narzędzi online do wygenerowania wszystkich wymaganych rozmiarów:
- https://realfavicongenerator.net/
- https://favicon.io/

## Kolory tematu:
- Theme color: `#1e40af` (niebieski - zgodny z motywem aplikacji)
- Background color: `#ffffff` (biały)

## Integracja:

Wszystkie linki do faviconów są już dodane w `index.html`:
- Linie 10-20 zawierają pełną konfigurację favicon
